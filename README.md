> Razzle, Express and typeorm boilerplate

## Prerequisites

- node 12.14.1
- yarn 1.21.1

## Install

```sh
git clone https://github.com/yasintz/retb.git
```

## Usage for development 

```sh
yarn install && yarn run dev 
```

## Run tests

```sh
yarn run test
```

## Usage for production 

```sh
yarn install && yarn run build && yarn run start 
```
